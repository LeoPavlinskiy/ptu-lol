# Проект расчёта параметров верхней панели крыла

Проект для расчёта параметров верхней панели крыла самолёта Boeing 737-800 с учётом местной и общей потери устойчивости обшивки.

## Структура проекта

```
пту/
├── wing-panel-design/     # Основной проект расчёта панели
│   ├── main.py            # Главный файл запуска
│   ├── README.md          # Документация проекта
│   └── ...                # Модули расчёта
├── docs/                  # Документация и исследования
│   ├── researches/        # Результаты исследований
│   ├── book/              # Книга Вольмира (локально)
│   └── ...                # Другая документация
└── tdl.md                 # Технический дизайн-лист
```

## Основной проект

Основной проект находится в директории `wing-panel-design/`. 

См. [wing-panel-design/README.md](wing-panel-design/README.md) для подробной информации о проекте расчёта панели.

## Быстрый старт

```bash
cd wing-panel-design
pip install -r requirements.txt
python main.py
```

## Описание

Проект реализует алгоритм подбора параметров верхней панели крыла (обшивка + стрингеры) с учётом:
- Местной потери устойчивости обшивки между стрингерами
- Местной потери устойчивости элементов стрингеров
- Общей потери устойчивости панели как сжатого стержня
- Редуцирования обшивки после потери устойчивости (метод Винтера)
- Проверки прочности по допускаемым напряжениям

## Технологии

- Python 3
- Материал: В95-Т1 (алюминиевый сплав)
- Самолёт: Boeing 737-800
