## 1. Класс самолёта

* **Класс:** пассажирский, магистральный, узкофюзеляжный лайнер (narrow-body twin-engine turbofan). ([Википедия][1])

## 2. Массы (737-800)

По открытым данным Boeing и обобщённым моделям: ([Википедия][1])

* **Максимальная взлётная масса (MTOW / MTOMdes):**
  ≈ **79 000 кг** (точнее 79 016 кг).
* **Типичная «нормальная» взлётная масса для расчётов**
  (коммерческая загрузка, не полный бак): обычно **65–72 т**. Для инженерных прикидок удобно взять **70 000 кг** (это допущение, не сертификационное значение).
* **Максимальная посадочная масса:** ≈ 66–70 т (в зависимости от конкретного подварианта, берётся из сертификационного листа; для грубых расчётов можно считать ~66 т). ([boeing.com][2])

## 3. Геометрия крыла 737-800

### Основные размеры

(для крыла без учёта увеличения размаха winglet):

* **Размах крыла b:** 34,32 м ([reference.wolfram.com][3])
* **Площадь крыла S (gross):** ≈ **124,6 м²** ([aircraftinvestigation.info][4])
* **Удлинение крыла AR = b² / S:**
  AR ≈ 9,45 (в справочниках обычно округляют до 9,44). ([Википедия][1])
* **Коэффициент сужения λ = c_tip / c_root:**
  c_root = 6,65 м, c_tip = 1,25 м → λ ≈ 0,19. ([reference.wolfram.com][3])

### Стреловидность, угол установки, диэдр

Из параметров модели Boeing 737-800 в SystemModeler: ([reference.wolfram.com][3])

* **Стреловидность по ¼ хорды** Λ₍c/4₎ ≈ 25° (0,4367 рад)
* По формуле для трапециевидного крыла получаем:

  * **Стреловидность по передней кромке** Λ_LE ≈ **28–28,5°**
* **Диэдр крыла γ:** ≈ 6° (0,1047 рад)
* **Угол установки крыла i_w:** ≈ **3°** (0,05236 рад) относительно оси фюзеляжа.

### Профиль крыла

По открытым данным (airfoiltools): ([airfoiltools.com][5])

* Используется фирменный **Boeing 737 midspan airfoil** (семейство BAC / модифицированный суперкритический профиль).
* Для midspan-профиля указано:

  * **Максимальная относительная толщина:** ~12,5%
  * Положение максимальной толщины: ~29,7% хорды.
* В реальности на 737-800 применяется **семейство профилей**: более толстый в корне и более тонкий к законцовке, но конкретные номера профилей Boeing не публикует.

### Хорды

Из той же модели: ([reference.wolfram.com][3])

* **Корневая хорда c_root:** 6,65 м
* **Концевая хорда c_tip:** 1,25 м
* Средняя аэродинамическая хорда (приблизительно):
  ( \text{MAC} \approx \frac{2}{3} c_\text{root}\frac{1+\lambda+\lambda^2}{1+\lambda} \approx 4,2\ \text{м} ) (оценка по трапециевидной формуле).

## 4. Конструктивно-силовая схема крыла 737-800

### Общая схема

По описаниям конструкций крыла семейства 737 (NG/MAX): ([Scribd][6])

* Крыло — **алюминиевое, двухпутное, fail-safe, двухлонжеронное**.
* **Wing box** (силовой кессон) образован:

  * фронтальным и задним **основными лонжеронами** (front & rear spars),
  * верхней и нижней обшивкой,
  * продольными стрингерами,
  * поперечными нервюрами.
* Кессон одновременно служит **интегральным топливным баком**.
* К лонжеронам крепятся элементы механизации: предкрылки, закрылки, интерцепторы, элероны. Шасси опирается на задний лонжерон через силовую балку.

### Количество основных лонжеронов

* **2 основных силовых лонжерона** (front spar и rear spar) во всём размахе крыла.
* Дополнительно ближе к задней кромке есть вспомогательные элементы (false spar / trailing edge structure) под механизацию, но они не несут основной изгибающий момент.

### Расположение лонжеронов (в долях хорды)

Точных процентных значений Boeing публично не даёт, но по литературе по крыльевым боксам транспортных самолётов и моделям 737-800: ([www.slideshare.net][7])

* Для крыльев похожего класса:

  * **Передний лонжерон:** ~12–20% хорды (0,12–0,20 c)
  * **Задний лонжерон:** ~55–75% хорды (0,55–0,75 c)
* В статье по интеграции winglet для 737-800 для законцовки принимают:

  * front spar ≈ 0,20 c
  * rear spar ≈ 0,74 c
* Для предварительных расчётов по 737-800 разумно принять:

  * у корня: front spar ~0,15 c, rear spar ~0,60 c
  * у законцовки: front spar ~0,20 c, rear spar ~0,74 c

**Важно:** это инженерная оценка по открытым источникам, а не официальные чертежи Boeing.

### Тип силового набора

Из описаний конструкторских руководств 737: ([Scribd][6])

* **Тип:** кессонное **лонжеронно-стрингерное** крыло:

  * обшивка (верх/низ) — листы из Al-сплава, усиленные продольными **стрингерами** (Z- или J-образного сечения);
  * лонжероны — балки с вертикальной стенкой и полками;
  * нервюры задают форму профиля и делят топливные отсеки.

### Высота силового кессона (по размаху)

По модельным данным: максимальная геометрическая толщина крыла 737-800: ([reference.wolfram.com][3])

* **Максимальная толщина профиля:**

  * у корня t_root ≈ 0,60 м
  * у законцовки t_tip ≈ 0,25 м
* Практически высота силового кессона (расстояние между основными несущими панелями) чуть меньше толщины профиля (минус обшивки, технологические зазоры). Для начальной оценки обычно берут **h_box ≈ 0,8·t**.
  Тогда ориентировочно:

  * у корня: **h_box ≈ 0,5 м**
  * на половине размаха: **h_box ≈ 0,35 м** (при t ≈ 0,45 м по интерполяции)
  * у законцовки: **h_box ≈ 0,2 м**

Эти значения — **оценочные**, но по порядку величины реалистичны для прочностных расчётов.

## 5. Распределение аэродинамической нагрузки по размаху

Точных экспериментальных «боинговских» диаграмм в открытом доступе нет, поэтому для расчёта прочности чаще используют **идеализированное эллиптическое распределение подъёмной силы** на полукрыле:

* Пусть полуразмах **L = b/2 = 17,16 м**.
* Полная подъёмная сила при 1g на MTOW:
  ( W = m g = 79,016 ,\text{кг} \cdot 9{,}81 \approx 7,75 \cdot 10^5 ,\text{Н} ).
  На одно полукрыло → **Lₕ ≈ 3,88·10⁵ Н**.
* Линейная нагрузка вдоль полуразмаха (эллиптическая):

[
q(y) = q_0 \sqrt{1 - \left(\frac{y}{L}\right)^2},\quad
q_0 = \frac{4 L_h}{\pi L}
]

где (y) — расстояние от плоскости симметрии (0…L).

Подставляя числа, получаем **q₀ ≈ 28,8 кН/м** в корне.

Типичные значения q(y) (1g, MTOW, одно полукрыло, без учёта веса крыла, двигателей и топливного распределения — т.е. чистая аэродинамическая схема):

| Положение по размаху | y, м | q(y), кН/м (прибл.) |
| -------------------- | ---- | ------------------- |
| Корень               | 0    | 28,8                |
| 0,25 L               | 4,29 | 26,5                |
| 0,5 L                | 8,58 | 22,1                |
| 0,75 L               | 12,9 | 14,4                |
| Законцовка           | 17,2 | ≈0                  |

Это даёт плавное, близкое к эллиптическому распределение, что обычно и принимают в предварительных прочностных расчётах.

## 6. Типичные изгибающие моменты в консоли крыла

Используем ту же идеализированную эллиптическую нагрузку (1g, MTOW, только подъёмная сила). Изгибающий момент в сечении на расстоянии y от фюзеляжа:

[
M(y) = \int_{y}^{L} (x - y), q(x), dx
]

Численно (per wing):

| Положение по размаху | y, м | M(y), Н·м | M(y), МН·м (округлённо) |
| -------------------- | ---- | --------- | ----------------------- |
| Корень (у борта)     | 0    | 2,82·10⁶  | **≈ 2,8 МН·м**          |
| 0,25 L               | 4,29 | 1,42·10⁶  | ≈ 1,4 МН·м              |
| 0,5 L                | 8,58 | 5,33·10⁵  | ≈ 0,53 МН·м             |
| 0,75 L               | 12,9 | 9,7·10⁴   | ≈ 0,10 МН·м             |
| Законцовка           | 17,2 | ≈ 0       | 0                       |

Замечания:

* Это **только аэродинамический вклад** при 1g и MTOW.
  Для полноценного расчёта прочности нужно добавить:

  * вес крыла, двигателей, топлива,
  * дополнительные нагрузки (манёвренные, порывовые, несимметричные),
  * коэффициенты запаса по нормам CS/FAR-25.
* Реальные расчёты для сертификации используют более сложные расчётные нагрузки (манёвры ±n, порывы, аварийные режимы и т.п.).
