# Техническое задание: Алгоритм подбора параметров верхней панели крыла

## 1. Общая информация

### 1.1 Цель проекта
Создать алгоритм на Python для подбора параметров верхней панели крыла большого удлинения с учётом требований статической прочности.

### 1.2 Область применения
- Расчёт прочности верхней панели крыла самолёта
- Подбор геометрических параметров панели
- Анализ устойчивости элементов конструкции
- Учёт редуцирования элементов при потере устойчивости

### Этап 3: Реализация алгоритма

#### 3.1 Последовательность расчёта для каждого сечения

1. **Ввод исходных данных:**
   - Параметры самолёта
   - Свойства материала
   - Номер расчётного сечения (0.2, 0.4, 0.6, 0.8)

2. **Расчёт геометрии сечения:**
   - Положение сечения по размаху
   - Геометрия крыла в данном сечении (хорда, высота профиля)
   - Габариты силового кессона
   - Площадь полок лонжеронов

3. **Расчёт нагрузок:**
   - Изгибающий момент в сечении (из бота Чедрика или расчёт)
   - Нормальная сила (если учитывается)

4. **Предварительный подбор параметров панели:**
   - Начальная толщина обшивки
   - Тип и количество стрингеров
   - Шаг стрингеров

5. **Расчёт критических напряжений:**
   - Местная потеря устойчивости обшивки
   - Местная потеря устойчивости рёбер стрингера
   - Общая потеря устойчивости панели

6. **Расчёт напряжений в панели:**
   - Напряжения от изгибающего момента
   - Учёт работы всех элементов (обшивка + стрингеры)

7. **Проверка устойчивости и редуцирование:**
   - Сравнение напряжений с критическими
   - Если напряжение превышает критическое - редуцирование элемента
   - Итерационный пересчёт с учётом редуцирования

8. **Проверка прочности:**
   - Сравнение расчётных напряжений с пределом пропорциональности
   - Если не выполняется - корректировка параметров панели

9. **Оптимизация (опционально):**
   - Подбор параметров для минимизации веса
   - Проверка всех ограничений

10. **Вывод результатов:**
    - Все требуемые параметры для данного сечения

#### 3.2 Алгоритм редуцирования обшивки

```
1. Расчёт критического напряжения обшивки σкр.обш
2. Расчёт действующего напряжения σдейств
3. Если σдейств > σкр.обш:
   a. Расчёт коэффициента редуцирования k = f(σдейств/σкр.обш)
   b. Расчёт эффективной ширины обшивки bэф = k * b
   c. Пересчёт приведённой площади панели с учётом bэф
   d. Пересчёт напряжений
   e. Проверка сходимости
   f. Если не сходится - повтор с шага 3a
4. Аналогично для стрингеров, если они потеряли устойчивость
```
