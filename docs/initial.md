Задача для зачёта по предмету Прикладная теория упругости в зимнюю сессию

Цель: Создать алгоритм для подбора параметров верхней панели крыла большого удлинения с учётом требований статической прочности.

Исходные данные:
(a) Параметры самолёта (выберите самостоятельно, можно использовать реальные ЛА):
1. Название самолёта.
2. Класс самолёта.
3. Взлётный вес.
4. Геометрия крыла.
5. Конструктивно-силовая схема крыла.

(b) Материал:
1. Алюминиевый сплав (например, В95).

Требования к расчётам:
(a) Для построения эпюр используйте бот Чедрика В.В. (@NagRuZkiBot).
(b) Выполните расчёт для четырёх сечений: 0,2, 0,4, 0,6, 0,8 длины консоли крыла.
(c) Для анализа прочности используйте формулы, изученные на семинарах.

Результаты должны включать:
(a) Действующий изгибающий момент в каждом сечении (бот Чедрика).
(b) Габариты силового кессона в данном сечении.
(c) Площадь полок лонжеронов (если имеются).
(d) Предел пропорциональности выбранного материала.
(e) Параметры панели:
1. Площадь панели.
2. Толщина обшивки.
3. Шаг стрингеров (их количество на панели).
4. Все геометрические параметры стрингера, в зависимости от выбранного типа.

(f) Расчётные напряжения в панели с учётом редуцирования элементов, потерявших устойчивость.
(g) Критические напряжения:
1. Местная потеря устойчивости обшивки.
2. Местная потеря устойчивости рёбер стрингера.
3. Общая потеря устойчивости панели.

Алгоритм должен быть реализован на языке программирования Python (учитывая его дальнейшее применение на курсе Балунова К.А.).
Хотя достижение весового совершенства панелей не является обязательным, к нему стоит стремиться.
Алгоритм должен учитывать редуцирование обшивки при её закритическом деформировании.
Для получения зачёта необходимо представить код алгоритма и объяснить его работу.
Источники: Вольмир А.С. «Устойчивость упругих систем» (1967), пример 9.3, 9.4, стр. 383.
