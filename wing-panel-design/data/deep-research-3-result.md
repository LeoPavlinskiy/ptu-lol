Ниже я дам набор формул и приёмов, которые используются у Вольмира в главах про прямоугольные и подкреплённые пластинки (§80–§82, §94–§95 в «Устойчивости упругих систем» / «Устойчивости деформируемых систем») и которые по сути стоят за расчётами примеров 9.3–9.4. Я не воспроизвожу буквально текст примеров, но использую те же модели и зависимые формулы. ([books.totalarch.com][1])

Для определённости считаем:

* панель прямоугольная, длина вдоль стрингеров — (a), ширина между шпангоутами — (b);
* шаг стрингеров — (s);
* толщина обшивки — (t);
* модуль упругости — (E), коэффициент Пуассона — (\nu);
* продольное сжимающее напряжение по обшивке — (\sigma_x) (по оси стрингеров).

---

## 1. Местная потеря устойчивости обшивки между стрингерами

Обшивка между стрингерами моделируется как прямоугольная пластинка шириной (b_p \approx s), толщиной (t), длинная в направлении сжатия ((a / b_p \gtrsim 3)).

Критическое напряжение при одноосном сжатии (классическая формула для пластинки Вольмира / Тимошенко): ([aquibzafar.files.wordpress.com][2])

[
\sigma_{\text{cr,skin}}
=======================

k_\sigma ,
\frac{\pi^2 E}{12(1-\nu^2)}
\left(\frac{t}{b_p}\right)^2
]

где:

* (k_\sigma) — коэффициент устойчивости (buckling coefficient) в зависимости от граничных условий по краям (y = 0,,b_p);
* (b_p) — расчётная ширина поля обшивки (обычно шаг стрингеров, реже — расстояние между осями стенок стрингеров); [м]
* (t) — толщина обшивки, [м];
* (E) — модуль Юнга, [Па];
* (\nu) — коэффициент Пуассона (для дюралюминия ~0.3).

Типичные (k_\sigma):

* **оба края шарнирно опёрты (обшивка «на шарнирах» по стрингерам)**
  (k_\sigma \approx 4.0) для длинной пластины. ([assets.w3.tue.nl][3])

* **оба края жёстко защемлены стрингерами**
  (k_\sigma \approx 6.97).

* **один край защемлён, другой шарнирно опёрт** — промежуточное значение, часто берут (k_\sigma \approx 5.0). ([ScienceDirect][4])

Форма потери устойчивости (первый мод) для длинной пластины:

[
w(x,y) = W \sin\frac{\pi x}{a},\sin\frac{\pi y}{b_p}
]

## 2. Местная потеря устойчивости элементов стрингера

Каждый элемент сечения стрингера (пояс, стенка) рассматривается как пластинка:

[
\sigma_{\text{cr,plate}} =
k_{\sigma,i},
\frac{\pi^2 E}{12(1-\nu^2)}
\left(\frac{t_i}{b_i}\right)^2
]

где:

* (b_i) — ширина элемента (вынос полки, высота стенки между полками);
* (t_i) — толщина элемента;
* (k_{\sigma,i}) — коэффициент устойчивости для конкретных условий закрепления.

Типичные случаи (локальная устойчивость тонких поясов и стенок, широко используемые и у Вольмира, и в современных нормах): ([research.chalmers.se][5])

1. **Пластинка с обоими длинными краями защемлёнными** (внутренняя часть стенки между полками, или пояс между вертикальными стенками):
   [
   k_{\sigma} \approx 6.97
   ]

2. **Внутренний сжатый пояс** (оба края опёрты/защемлены):
   [
   k_{\sigma} \approx 4.0\text{–}6.0
   ]

3. **Свободный вынос полки (консольная пластинка: один край жёстко связан с стенкой, другой свободен)** — классический случай «outstand flange»:
   [
   k_{\sigma} \approx 0.43
   ]

4. **Пластинка, опёртая на трёх сторонах и свободная по четвёртой** (напр. пояс Z-стрингера):
   [
   k_{\sigma} \approx 0.425
   ]

Проверка местной устойчивости стрингера:
для каждого элемента сечения вычисляете (\sigma_{\text{cr,plate}}) и сравниваете с рабочими напряжениями.

## 3. Общая потеря устойчивости панели (как целого элемента)

Общая потеря устойчивости панели при осевом сжатии обычно рассчитывается как потеря устойчивости эквивалентного стержня (колонны), составленного из стрингеров и **эффективной** части обшивки.

### 3.1. Эффективные площадь и момент инерции

Перед общей устойчивостью (если обшивка уже частично потеряла местную устойчивость) используют **редуцированную ширину** обшивки (b_{\text{eff}}) (см. §6).

Тогда:

* эффективная площадь:
  [
  A_\text{eff} = A_s + t,b_{\text{eff}}
  ]
* момент инерции относительно нейтральной оси панели:
  [
  I_\text{eff} = I_s + t,b_{\text{eff}}\left(\frac{t^2}{12} + z_0^2\right)
  ]
  где (z_0) — расстояние от середины обшивки до нейтральной оси секции (с учётом стрингера).

(Если несколько стрингеров — суммируете по всем, деля на шаг и работая «на 1 шаг» панели.)

### 3.2. Эйлеровская формула с учётом граничных условий

Критическая сила:

[
N_{\text{cr,glob}} = \frac{\pi^2 E_\text{red} I_\text{eff}}{(\mu L)^2}
]

где:

* (L) — длина панели по оси сжатия;
* (\mu) — коэффициент приведённой длины (зависит от закрепления по концам);
* (E_\text{red}) — редуцированный модуль (см. §7).

Критическое среднее напряжение по **эффективной** площади:

[
\sigma_{\text{cr,glob}} = \frac{N_{\text{cr,glob}}}{A_\text{eff}} =
\frac{\pi^2 E_\text{red}}{\lambda_\text{eff}^2}
]

где

[
\lambda_\text{eff} = \frac{\mu L}{r_\text{eff}}, \qquad
r_\text{eff} = \sqrt{\frac{I_\text{eff}}{A_\text{eff}}}
]

— эффективная гибкость панели.

## 4. Учёт граничных условий

### 4.1. Для пластинок (обшивка, элементы стрингера)

Граничные условия входят в коэффициент устойчивости (k_\sigma).

Основные случаи для одноосного сжатия:

| Условие по длинным краям            | (k_\sigma) (длинная пластинка) |
| ----------------------------------- | ------------------------------ |
| Оба края шарнирно опёрты            | ~4.0                           |
| Оба края защемлены                  | ~6.97                          |
| Один край защемлён, другой шарнирно | ~5.0                           |
| Один край защемлён, другой свободен | ~0.43                          |

Эти значения совпадают по порядку с таблицами у Вольмира и современными обзорными работами по устойчивости пластин. ([aquibzafar.files.wordpress.com][2])

### 4.2. Для панели как стержня

Коэффициент приведённой длины (\mu) в формуле Эйлера:

* оба конца шарнирно опёрты: (\mu = 1.0);
* один конец защемлён, другой шарнирно: (\mu = 0.7);
* оба конца защемлены: (\mu = 0.5);
* один конец свободный, другой защемлён: (\mu = 2.0).

## 5. Коэффициенты устойчивости для разных типов стрингеров

Стрингер (Z, швеллер, тавр и т.п.) раскладывается на **элементы-пластинки**:

* вертикальная стенка;
* пояса (внутренние и выносные);
* возможные косынки.

Для каждого элемента:

[
\sigma_{\text{cr},i} =
k_{\sigma,i},
\frac{\pi^2 E}{12(1-\nu^2)}
\left(\frac{t_i}{b_i}\right)^2
]

Типичные (k_{\sigma,i}) в зависимости от формы:

* **Тавровый стрингер**

  * стенка между полкой и обшивкой — оба края защемлены: (k\approx 6.97);
  * пояс (если внутренняя часть между двумя стенками/ребрами) — оба края опёрты: (k\approx 4\text{–}6);
  * свободный вынос полки — (k\approx 0.43).

* **Швеллерный стрингер (C-образный)**

  * стенка — как выше;
  * два выносных пояса (как консольные пластинки): (k\approx 0.43) для каждого.

* **Z-образный стрингер**

  * один пояс внутренний (опёртый/защемлённый) — (k\approx 4\text{–}6);
  * другой — выносной (консоль) — (k\approx 0.43).

В авиационных расчётных рекомендациях часто приводят специальные таблицы предельных отношения (b/t) для каждой формы — по сути это то же самое, но записанное как ограничение на тонкостенность при заданном (\sigma). ([steelconstruction.info][6])

## 6. Эффективная ширина обшивки при закритическом деформировании

После местной потери устойчивости обшивка несёт нагрузку в основном в узких полосах около стрингеров. Это описывается через **эффективную ширину** (b_{\text{eff}}), меньшую реальной ширины (b_p).

Классический подход (фон Карман, затем Винтер и др.): ([ascjournal.com][7])

1. Сначала определяется **эластическое** критическое напряжение (\sigma_{\text{cr,skin}}) по формуле п.1.

2. Вводится безразмерная «тонкостенность» (псевдо-гибкость элемента):

   [
   \bar\lambda_p = \sqrt{\frac{\sigma_y}{\sigma_{\text{cr,skin}}}}
   ]

   где (\sigma_y) — условный предел текучести материала.

3. Эффективная ширина:

   * если (\bar\lambda_p \le 0.673) → пластинка работает практически полностью,
     [
     b_{\text{eff}} = b_p
     ]
   * если (\bar\lambda_p > 0.673) (тонкая/гибкая пластинка),
     [
     \rho = \frac{b_{\text{eff}}}{b_p} =
     \frac{1 - 0.22/\bar\lambda_p}{\bar\lambda_p}
     ,\qquad
     b_{\text{eff}} = \rho,b_p
     ]

Эта зависимость (вариант формулы Винтера) по сути соответствует тем же идеям, что у Вольмира при рассмотрении закритической работы пластинок и панелей; просто в современных нормах она зашита в виде эмпирических коэффициентов. ([ascjournal.com][7])

## 7. Редуцированный модуль упругости

Вольмир рассматривает панель в упруго-пластической и закритической области через **редуцирование жёсткости**:

* часть обшивки находится в посткритическом состоянии,
* напряжения перераспределяются,
* общая продольная жёсткость панели уменьшается.

В терминах общей продольной жёсткости панели:

[
N_x = A_{\text{eff}} E_{\text{red}} , \varepsilon_x
]

где (N_x) — продольная сила на шаг (s), (\varepsilon_x) — средняя продольная деформация.

Типичный инженерный способ задать (E_{\text{red}}):

[
E_{\text{red}} =
\frac{E A_s + E_{\text{skin,eff}}, A_{\text{skin,eff}}}
{A_s + A_{\text{skin,eff}}}
]

* (A_s) — суммарная площадь стрингера(ов) на шаг;
* (A_{\text{skin,eff}} = t,b_{\text{eff}});
* (E_{\text{skin,eff}} \le E) — «эффективный» модуль обшивки (учитывает, что часть поля работает в мембранно-посткритическом режиме).

Если по диаграмме «напряжение–деформация» известен касательный модуль (E_t) в области текущих напряжений, то для стержневых задач часто применяют **теорию редуцированного модуля Энгессера–Карман**: ([projekter.aau.dk][8])

[
E_r = \frac{E I_1 + E_t I_2}{I_1+I_2},\qquad
\sigma_{\text{cr}} = \frac{\pi^2 E_r}{\lambda^2}
]

где (I_1) и (I_2) — моменты инерции частей сечения в зонах разгрузки (упругой) и догрузки (с касательным модулем (E_t)). Для самолётных панелей обычно ограничиваются более простой схемой через (b_{\text{eff}}) и считают (E_t \approx E) до близости к текучести.

## 8. Взаимодействие обшивки и стрингеров при потере устойчивости

Картина (как у Вольмира для подкреплённых панелей):

1. **До местной потери устойчивости**

   * обшивка и стрингеры работают совместно;
   * напряжения распределены примерно равномерно по сечению;
   * общая жёсткость определяется полной площадью обшивки и стрингеров.

2. **Местная потеря устойчивости обшивки между стрингерами**

   * в полях обшивки образуется волнистая форма (локальное выпучивание);
   * средние напряжения по большой части ширины падают ниже уровня до потери устойчивости;
   * «несущая зона» оказывается сосредоточена в полосах вокруг стрингеров — это и есть эффективная ширина (b_{\text{eff}}).

3. **Рост нагрузки**

   * обшивка в пределах (b_{\text{eff}}) и стрингеры продолжают нести возрастающую нагрузку;
   * общий модуль (E_{\text{red}}) падает;
   * возможно последующее **местное** выпучивание элементов стрингера (полок, стенки).

4. **Общая потеря устойчивости панели**

   * когда продольная сила достигает Эйлеровской (с учётом (E_\text{red}) и (I_\text{eff})), панель как целое теряет устойчивость (глобальный изгиб, иногда с комбинированными формами локальных гофров).

Это и есть та иерархия «местная обшивка → местная стрингер → общая панель», которую аккуратно описывает Вольмир в §94–§95 / §109–§110. ([books.totalarch.com][1])

## 9. Итерационный расчёт с учётом редуцирования (по духу Вольмира)

Типичная последовательность (она же лежит за «методом редукционных коэффициентов» и процедурой последовательных приближений, которые в авиакурсах прямо привязывают к Вольмиру). ([ppt-online.org][9])

1. **Исходное приближение**

   * принимаете, что вся обшивка работает полностью: (b_{\text{eff}}^{(0)} = b_p);
   * (E_\text{red}^{(0)} = E);
   * считаете общую критическую силу (N_{\text{cr,glob}}^{(0)}) по Эйлеру.

2. **Проверка местной устойчивости обшивки**

   * по текущему напряжению (\sigma^{(0)}) в обшивке (например, по среднему напряжению при заданной внешней нагрузке, если она ниже глобальной) считаете (\sigma_{\text{cr,skin}});
   * если (\sigma^{(0)} \le \sigma_{\text{cr,skin}}) → обшивка упруга, (b_{\text{eff}}^{(1)} = b_p), можно остановиться;
   * если (\sigma^{(0)} > \sigma_{\text{cr,skin}}) → вводите (b_{\text{eff}}^{(1)}) по формуле Винтера/Кармана (см. §6).

3. **Редуцирование жёсткости**

   * пересчитываете площадь и момент инерции: (A_\text{eff}^{(1)}, I_\text{eff}^{(1)});
   * задаёте (E_\text{red}^{(1)}) (либо как взвешенную по площади величину, либо по более точной схеме с (E_t));
   * вычисляете новое (N_{\text{cr,glob}}^{(1)}).

4. **Уточнение напряжений**

   * при заданной внешней нагрузке (N) получаете новые средние напряжения в обшивке и стрингерах:
     [
     \sigma_{x,\text{skin}}^{(1)} \approx \frac{N}{A_\text{eff}^{(1)}} \cdot \frac{t b_{\text{eff}}^{(1)}}{A_\text{eff}^{(1)}} \quad\text{и т.п.}
     ]
   * по ним снова считаете (\bar\lambda_p) и, если надо, корректируете (b_{\text{eff}}).

5. **Повторение итераций**

   * повторяете шаги 2–4 до тех пор, пока изменения (b_{\text{eff}}), (E_\text{red}) и (\sigma) не станут малы (на практике 2–3 итераций обычно достаточно).

6. **Контроль местной устойчивости стрингера**

   * на финальном шаге проверяете местную устойчивость его элементов по формулам из §2;
   * при необходимости вводите **дополнительное** редуцирование (уменьшаете эффективную ширину полок, повышаете гибкость и т.д.).
