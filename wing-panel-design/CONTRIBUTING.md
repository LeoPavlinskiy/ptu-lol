# Руководство по внесению вклада

## Структура проекта

Проект организован в модули:
- `aircraft_data.py` - данные о самолёте
- `material.py` - свойства материала
- `geometry.py` - расчёт геометрии
- `loads.py` - расчёт нагрузок
- `panel.py` - класс панели
- `stringer.py` - класс стрингера
- `stability.py` - расчёт устойчивости
- `reduction.py` - редуцирование обшивки
- `strength.py` - проверка прочности
- `output.py` - вывод результатов

## Стиль кода

- Используйте docstrings для всех функций и классов
- Следуйте PEP 8
- Добавляйте комментарии к сложным формулам
- Указывайте источники формул в комментариях

## Тестирование

Перед отправкой изменений:
1. Запустите все тесты: `python test_*.py`
2. Запустите проверку физической корректности: `python validate_physics.py`
3. Убедитесь, что основной расчёт работает: `python main.py`

## Источники формул

Формулы должны иметь ссылки на источники:
- Вольмир А.С. "Устойчивость деформируемых систем"
- deep-research-3-result.md - методы расчёта устойчивости
