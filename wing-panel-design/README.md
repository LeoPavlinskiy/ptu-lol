# Расчёт параметров верхней панели крыла

Программа для расчёта параметров верхней панели крыла самолёта Boeing 737-800 с учётом местной и общей потери устойчивости обшивки.

## Описание

Проект реализует алгоритм подбора параметров верхней панели крыла (обшивка + стрингеры) с учётом:
- Местной потери устойчивости обшивки между стрингерами
- Местной потери устойчивости элементов стрингеров
- Общей потери устойчивости панели как сжатого стержня
- Редуцирования обшивки после потери устойчивости (метод Винтера)
- Проверки прочности по допускаемым напряжениям

## Использование

Запустите главный файл:
```bash
python main.py
```

Программа выполнит расчёт для четырёх сечений по размаху крыла (z/L = 0.2, 0.4, 0.6, 0.8) и сохранит результаты в файл `data/results.md`.

## Основные модули

### aircraft_data.py
Класс `Aircraft` для хранения геометрических параметров Boeing 737-800:
- Хорда крыла по размаху
- Высота кессона
- Положение лонжеронов
- Ширина кессона

### material.py
Класс `Material` для свойств алюминиевого сплава В95-Т1:
- Предел прочности: 520 МПа
- Предел текучести: 440 МПа
- Предел пропорциональности: 352 МПа
- Модуль упругости: 70 ГПа

### panel.py
Класс `Panel` для представления панели в сечении:
- Геометрические параметры (ширина, высота, толщина обшивки)
- Стрингеры
- Эффективные параметры после редуцирования

### stringer.py
Класс `Stringer` для моделирования стрингеров типов Z, C, T, L:
- Геометрические параметры
- Площадь поперечного сечения
- Момент инерции

### stability.py
Функции расчёта критических напряжений:
- `local_skin_buckling()` - местная потеря устойчивости обшивки
- `local_stringer_buckling()` - местная потеря устойчивости элементов стрингеров
- `general_panel_buckling()` - общая потеря устойчивости панели

### reduction.py
Методы редуцирования обшивки:
- `effective_width()` - эффективная ширина обшивки (методы Винтера и фон Кармана)
- `reduced_modulus()` - редуцированный модуль упругости
- `iterative_reduction()` - итерационный расчёт с учётом редуцирования

### strength.py
Расчёт напряжений и проверка прочности:
- `calculate_stresses()` - расчёт напряжений в панели
- `check_strength()` - проверка прочности по допускаемым напряжениям
- `check_panel_strength()` - полная проверка прочности панели

## Алгоритм расчёта

1. **Инициализация**: Загрузка данных о самолёте, материале и нагрузках
2. **Геометрия**: Расчёт геометрических параметров для каждого сечения
3. **Подбор параметров**: Предварительный подбор толщины обшивки и стрингеров
4. **Итерационный расчёт**:
   - Расчёт критических напряжений местной потери устойчивости
   - Расчёт действующих напряжений
   - Редуцирование обшивки при превышении критических напряжений
   - Пересчёт эффективных параметров
   - Повторение до сходимости
5. **Проверка прочности**: Сравнение напряжений с допускаемыми значениями
6. **Вывод результатов**: Сохранение результатов в markdown файл
